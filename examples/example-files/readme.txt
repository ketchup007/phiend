These files are provided for your convenience and can be used
as a base for configuring your site.

bootstrap.php
	The only entry point to your application, i.e. the file
	users will request by typing its name in their browser.
	Rename it to what you like and make sure the paths
	defined inside this file are correct.

phiend-config.xml
	The only configuration file for phiend.
	It cannot be renamed and must be placed in
	{CODE_DIR}/config/phiend.config.xml, where {CODE_DIR}
	is whatever you specify in bootstrap.php.
